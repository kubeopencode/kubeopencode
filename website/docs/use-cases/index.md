# Use Cases

Real-world examples of how to use KubeOpenCode for common development workflows.

- **[Docker in Docker](docker-in-docker.md)** - Run Docker inside agent containers (Sysbox, privileged, rootless)
- **[VS Code in Browser](vscode-in-browser.md)** - Run a full VS Code editor alongside your AI agent, accessible from any browser
- **[Sharing Agent Terminals](agent-share-link.md)** - Share terminal access via URL for QA, demos, and external collaboration
